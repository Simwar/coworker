# Running coworker on Astro

Open-source AI agent with MCP UI, app builder, A2A protocol, skills marketplace, and multi-provider chat

## Prerequisites

- **[Astro CLI (`ast`)](https://astropods.ai)** — install from https://astropods.ai
- **Docker Desktop** — required for local development
- API key(s) for the model provider (see [Configuration](#configuration) below)

---

## Quick start

### 1. Install the ast CLI

Follow the installation instructions at **https://astropods.ai**.

Verify the installation:

```bash
ast --version
```

### 2. Configure credentials

```bash
ast configure
```

This interactive prompt saves your API keys securely. You will need:

| Variable | Description |
|----------|-------------|
| `ANTHROPIC_API_KEY` | Anthropic Claude API key |

### 3. Run the agent locally

```bash
ast dev
```

Open the playground at **http://localhost:3000** and start chatting with the agent.

Tail the logs in a separate terminal:

```bash
ast dev logs
```

### 4. Deploy to Astro

```bash
ast push
```

## How it works

The agent is built with [Mastra](https://mastra.ai) and connected to the Astro messaging infrastructure via `@astropods/adapter-mastra`. The `astropods.yml` file declares the agent container, model provider, and integrations. `ast dev` orchestrates the containers locally; `ast push` builds and deploys to the Astro cloud.
